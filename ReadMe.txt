Steps to run the website

1. download and install NuGet plugin to Microsoft Visual Studio
   for more information visit http://cloudinary.com/documentation/dotnet_integration

2. Install cloudinaryDotNet package to visual studio(also given in the above mentioned site)

3.Import the project to your workspace

4.Make sure you are connected to the internet

4.Now you are good to go.... run the website!
